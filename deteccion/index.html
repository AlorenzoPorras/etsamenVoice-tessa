Welcome

I am a virtual assistant that can help you operate different electronic devices. You can interact with me in the following way. The following voice commands are the keywords to use. Please note that you must say "2B" before each action.

Voice Command - Turn on/off bedroom light - Turns on/off the bedroom light (1 bulb)

Voice Command - Turn on/off living room light - Turns on/off the living room light (1 bulb)

Voice Command - Turn on/off garden lights - Turns on/off the garden lights (5 bulbs)

Voice Command - Turn on/off the fan - Turns on/off the ceiling fan (1 ceiling fan)

Voice Command - Open/close the curtains - Opens/closes the curtains (3 curtains)

Voice Command - Activate/deactivate the alarm - Activates/deactivates the house alarm (1 alarm with siren)

Voice Command - Turn on/off the cameras - Turns on/off the security cameras (3 surveillance cameras)

Voice Command - Go to monitoring - Redirects to the monitoring page
